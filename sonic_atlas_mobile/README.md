# Sonic Atlas Mobile

Android-first Expo / React Native mobile app for Sonic Atlas.

This workspace lives at:

`/Users/adamdavis/Desktop/OuterTune-dev/sonic_atlas/sonic_atlas_mobile`

The initial scaffold is aligned to `../sonic_mobileapp_plan.md`:

- Expo Router app shell
- Bottom tabs: Discover, Search, Playlists, My Atlas, Now Playing
- Shared Sonic Atlas dark/orange design tokens
- Live API client pointed at `https://sonic-atlas.vercel.app`
- Starter player shell for the Phase 1 now playing experience

## Next steps

1. Install dependencies in this directory.
2. Run `npm run start` or `npx expo start`.
3. Add mobile auth support to the backend so `My Atlas` can use protected routes.
4. Replace the placeholder player store with `react-native-track-player`.
